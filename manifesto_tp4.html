Resumo:

Este trabalho envolve a criação de um servidor Node.js capaz de redirecionar utilizadores para as diversas páginas do Arquivo Musical Digital criado em aulas anteriores.

O sistema deve ser capaz de reconhecer se o utilizador quer o index do arquivo ou uma obra, e caso queira a obra qual obra é que o utilizador precisa de receber exatamente.
